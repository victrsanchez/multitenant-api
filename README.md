## Description

Midi Multitenant Graphql Api with NestJs.

## Installation

```bash
$ npm install
```

## Set Databases

```bash
# Main database
Create a database in Postgres
$ create a .env using the .env.example with your credentials

# Tenant example databse
Create a database in Postgres for one tenant

```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
